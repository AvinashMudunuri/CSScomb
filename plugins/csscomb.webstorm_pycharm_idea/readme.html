How to install CSScomb for IntelliJIDEA/WebStorm/PyCharm

Required

  1. PHP
  2. CSScomb for IntelliJIDEA/WebStorm/PyCharm
  3. IntelliJIDEA or WebStorm or PyCharm

Attention! If you use Microsoft Windows you need to install PHP and set the $PATH.

Install CSScomb

  1. Open Preferences → External Tools
    Click Add
    • Name: CSScomb
    • Description: CSS normalizer
    • Program: php
    • Parameters: /path/to/csscomb.php $FilePath$
