How to install CSScomb for IntelliJ IDEA or WebStorm

Required

  1. PHP
  2. CSScomb for IntelliJIDEA/WebStorm
  3. IntelliJIDEA or WebStorm

Install

  1. Open Preferences → External Tools
    Click Add
    • Name: CSScomb
    • Description: CSS normalizer
    • Program: php
    • Parameters: /path/to/csscomb.php $FilePath$
