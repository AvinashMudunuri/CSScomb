Greatest tools bundle for sorting CSS properties in specific order

Features

  • Cортировка CSS-свойств Порядок свойств в помощь профессионалу
  • Настройка порядка CSS-свойств Используйте тот порядок, к которому вы привыкли
  • Разделение CSS-свойств на группы Группы свойств могут быть разделены пустой строкой
  • Полная поддержка CSS2/CSS2.1/CSS3 CSScomb готов к самому продвинутому CSS-коду
  • Парсинг CSS в тегах <style>, аттрибутах style="..." CSScomb найдет CSS среди других языков и отсортирует его.
  • Поддержка LESS/SASS Вложенные блоки, переменные и остальное будут отсортированы

Versions

  • CSScomb for Coda
  • CSScomb for IntelliJ IDEA or WebStorm
